# Eclipse getStringLength Plugin

Eclipse plugin that add the selected text counter on the right side of the column and row position in the Eclipse status bar.
After you double click the text, this plug-in will show the length of text you selected in the eclipse status bar; in the case you select more than one row it will show the number of rows selected, like Notepad++ do.

Eclipse MarketPlace : https://marketplace.eclipse.org/content/getstringlength
